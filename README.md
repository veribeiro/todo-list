<h3>üí° About the project</h3>

The to-do list website has a "lofi" main theme, designed to help users make task completion more productive.

In addition to the to-do list functionality, it also has an extra feature: a music player that plays lofi music, creating a more comfortable environment for the user.

> The goal of this project is to expand my knowledge, mainly in the JavaScript language, including what I had already learned in my technical course in Systems Development.

##

<h3>üñ•Ô∏è Features</h3>
<h4>To-do list:</h4>

* Time information, including whether it is morning, afternoon, or night (select the time already displayed on your device);

* Task addition field;

* Button that allows the user to show whether the task is completed;

* Button that allows the user to edit the task name (mainly used to correct what was written);

* Button to delete the task.

<h4>Music player:</h4>
(The songs in this music player have already been pre-selected)

* The image representing the artist's song;

* The song title, along with the artist's name;

* Button to start playing the song;

* Button to go back to the previous song;

* Button to skip to the next song;

* Progress bar;

* Delete the entire music player;

* Button to add the music player if it is deleted.
